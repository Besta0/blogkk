# Screenshots 截图文件夹

## 📸 如何添加截图

请将以下5张截图保存到此文件夹中：

1. **home.png** - 首页截图
   - 展示主页的 Hero 部分
   - 包含标题、简介和技能标签

2. **about.png** - 关于页面截图
   - 展示技能进度条
   - 展示工作经验时间线

3. **projects.png** - 项目展示截图
   - 展示项目卡片网格
   - 包含筛选标签

4. **blog.png** - 博客页面截图
   - 展示博客文章列表
   - 包含搜索和标签筛选

5. **contact.png** - 联系页面截图
   - 展示联系表单
   - 展示社交媒体链接

## 📝 命名规范

- 使用小写字母
- 使用 `.png` 格式（推荐）或 `.jpg` 格式
- 文件名必须完全匹配上述名称

## ✅ 完成后

添加完截图后，运行以下命令提交到 GitHub：

```bash
chmod +x commit-and-push.sh
./commit-and-push.sh
```

或者手动提交：

```bash
git add screenshots/
git commit -m "docs: Add screenshots"
git push origin main
```
